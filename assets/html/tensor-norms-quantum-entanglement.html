Classical Notion of Independence

Consider random variables $X,Y$ with finite support.

Definition

$X, Y$ are independent if for $x_i \in \operatorname{Ran}(X)$, $y_j \in \operatorname{Ran}(Y)$, $$ P(X=x_i,\,Y=y_j) = P(X=x_i)\,P(Y=y_j). $$ Otherwise they are dependent.

Goal

Given the matrix $M_{ij}=P(X=x_i, Y=y_j)$ decide whether $X$ and $Y$ are independent.

Observation

If $X$ and $Y$ are independent then $M = \mathbf u\mathbf v^{\!\top}$ where $$\mathbf u=(P(X=x_1),\dots)^{\!\top}, \quad \mathbf v=(P(Y=y_1),\dots)^{\!\top},$$ so $\operatorname{rank}(M)=1$.

Theorem

$X$ and $Y$ are independent $\iff \operatorname{rank}(M)=1$.

Proof. The forward direction is the previous observation. For the reverse direction, if $\operatorname{rank}(M)=1$ then $M=\mathbf u\mathbf v^{\!\top}$. Because $M_{ij}\ge0$ we can choose $u_i,v_j\ge0$ and set $$U=\sum_i u_i,\;V=\sum_j v_j.$$ One checks that $P(X=x_i)=u_iV$, $P(Y=y_j)=v_jU$, and $UV=1$, giving $$P(X=x_i)P(Y=y_j)=u_iV v_jU=u_i v_j( UV)=M_{ij}.\;\square$$

Entanglement of Pure States

Let $\mathcal H_X = \operatorname{span}\{|x_1\rangle,\dots,|x_m\rangle\}$ where $m=|\operatorname{Ran}(X)|$.
Let $\mathcal H_Y = \operatorname{span}\{|y_1\rangle,\dots,|y_n\rangle\}$ where $n=|\operatorname{Ran}(Y)|$.
Define the state vector corresponding to $M$ as $$|\psi\rangle = \sum_{i,j}\sqrt{P(X=x_i, Y=y_j)}\,|x_i\rangle\otimes|y_j\rangle.$$

Observation

$M=UV^{\!\top}$ iff $|\psi\rangle = |\psi_X\rangle\otimes|\psi_Y\rangle$ with $$|\psi_X\rangle=\sum_i\sqrt{P(X=x_i)}\,|x_i\rangle,\qquad|\psi_Y\rangle=\sum_j\sqrt{P(Y=y_j)}\,|y_j\rangle.$$

Definition (Tensor‑rank decomposition)

For Hilbert spaces $\mathcal H_1,\dots,\mathcal H_k$, a vector $|\psi\rangle\in\mathcal H_1\otimes\cdots\otimes\mathcal H_k$ admits a decomposition $$|\psi\rangle = \sum_{a=1}^r |\varphi_1^{(a)}\rangle\otimes|\varphi_2^{(a)}\rangle\otimes\cdots\otimes|\varphi_k^{(a)}\rangle.$$

Definition (Tensor rank)

$$R(|\psi\rangle)=\min\bigl\{r\in\mathbb N\,:\,|\psi\rangle\text{ admits a decomposition with }r\text{ simple tensors}\bigr\}.$$

  • $R(|\psi\rangle)=1$ $\iff$ $|\psi\rangle$ is a separable pure state.
  • $R(|\psi\rangle)>1$ $\iff$ $|\psi\rangle$ is entangled.

Examples (computing the tensor rank)

① Bipartite states ($k=2$)

If $|\psi\rangle = \sum_{ij}\Psi_{ij}|i\rangle\otimes|j\rangle$, then $R(|\psi\rangle)=\operatorname{rank}(\Psi)$ (the Schmidt rank).

Caution

For the classical‑to‑quantum mapping above, $R(|\psi\rangle)$ computed from $\sqrt{M_{ij}}$ may differ from $\operatorname{rank}(M)$.

Singular‑value decomposition

For a bounded operator $M$ on $\mathcal H^d$ there exist ONBs $\{|\alpha_i\rangle\}$, $\{|\beta_i\rangle\}$ and singular values $\sigma_1\ge\cdots\ge\sigma_r>0$ such that $$M=\sum_{i=1}^r\sigma_i|\alpha_i\rangle\langle\beta_i|.$$

② Example: $|\mathrm{GHZ}\rangle=\tfrac{1}{\sqrt2}(|000\rangle+|111\rangle)$

$R(|\mathrm{GHZ}\rangle)\le2$. To show $R(|\mathrm{GHZ}\rangle)>1$ assume, for contradiction, that $|\mathrm{GHZ}\rangle$ is separable:

$$|\mathrm{GHZ}\rangle = |\phi_A\rangle\otimes|\phi_B\rangle\otimes|\phi_C\rangle.$$ Then $$\rho_{\mathrm{GHZ}} = |\mathrm{GHZ}\rangle\langle\mathrm{GHZ}| = \rho_A\otimes\rho_B\otimes\rho_C,$$ with each $\rho_S$ pure (rank 1). Consider the partial transpose on $C$: $(\mathrm{Id}_{AB}\otimes T_C)(\rho_{\mathrm{GHZ}})=\rho_A\otimes\rho_B\otimes\rho_C^{\!T}$, which is positive of rank 1.

But for the true GHZ state, $$\rho_{\mathrm{GHZ}}=\frac12\bigl(|000\rangle\langle000|+|000\rangle\langle111|+|111\rangle\langle000|+|111\rangle\langle111|\bigr).$$ Taking $T_C$ one obtains a matrix with eigenvalues $\{\tfrac12,\tfrac12,\tfrac12,-\tfrac12,0,0,0,0\}$, hence not positive. Therefore $|\mathrm{GHZ}\rangle$ is entangled and $R(|\mathrm{GHZ}\rangle)=2$.

Remark

The tensor rank behaves like a “discrete norm” except that $R(\lambda|\psi\rangle)=R(|\psi\rangle)$ for any non‑zero scalar $\lambda$.

Entanglement of Mixed States

Let $\rho\in\mathcal M_d^{\mathrm{sa}}(\mathbb C)$ with $\rho\ge0$ and $\operatorname{Tr}(\rho)=1$.

Separable mixed state

$$\rho = \sum_i p_i\,\rho_i^{(1)}\otimes\rho_i^{(2)}\otimes\cdots\otimes\rho_i^{(k)},\qquad p_i>0,\;\sum_i p_i=1,$$ with each $\rho_i^{(j)}$ a density matrix on $\mathbb C^{d_j}$.

Clarifying notation

The product $\rho_i^{(1)}\otimes\rho_i^{(2)}$ is the tensor product of operators; if $\rho_i^{(1)}=|\psi\rangle\langle\psi|$ then $$\rho_i^{(1)}\otimes\rho_i^{(2)} = (|\psi\rangle\otimes|\phi\rangle)(\langle\psi|\otimes\langle\phi|).$$

Desirable properties of an entanglement measure $\Phi$

We want $\Phi$ convex so that for a separable state $\rho=\sum_a p_a\rho_a^{(1)}\otimes\cdots\otimes\rho_a^{(k)}$ we have $$\Phi(\rho)\le\sum_a p_a\,\Phi\bigl(\rho_a^{(1)}\otimes\cdots\otimes\rho_a^{(k)}\bigr).$$ If moreover $\Phi$ is normalised so that $\Phi$ of any product state is $\le C$ (often $C=1$), then $\Phi(\rho)\le C$ for separable $\rho$. Hence $\Phi(\rho)>C$ certifies entanglement.

Convex envelope

For $f:C\to\mathbb R$ on a convex $C$, its convex envelope is the largest convex lower bound $g\le f$ on $C$.

Theorem (rank vs. nuclear norm)

The convex envelope of $\operatorname{rank}(A)$ on $\{A:\|A\|_{S_\infty}\le1\}$ is the nuclear norm $\|A\|_{S_1}=\sum_i\sigma_i(A)$.

Lemma (variational form of $\|\cdot\|_{S_1}$)

$$\|A\|_{S_1}=\sup_{\|X\|_{S_\infty}\le1}|\operatorname{Tr}(A^\dagger X)|=\max_{U\text{ unitary}}|\operatorname{Tr}(AU)|.$$

Theorem (PPT test via nuclear norm)

Let $\rho$ be a bipartite state. If $\|\rho^{\Gamma_B}\|_{S_1}>1$ then $\rho$ is entangled.

Sketch. For separable $\rho$ we have $\rho^{\Gamma_B}\ge0$ and $\operatorname{Tr}(\rho^{\Gamma_B})=1$, hence $\|\rho^{\Gamma_B}\|_{S_1}=1$. Thus any value $>1$ witnesses entanglement.
