Tensor Norms for Quantum Entanglement

(Part 1)
Keywords: Quantum Entanglement, Tensor Norms, Separability, Nuclear Norm, Partial Transpose, PPT Criterion, Unextendible Product Basis

Abstract

This document introduces the fundamental concepts of quantum states and entanglement. We begin by defining pure, mixed, and separable states, establishing a framework for understanding composite quantum systems. The core challenge addressed is the detection of entanglement in a given quantum state. We demonstrate that while simple measures like matrix rank are sufficient for pure states, they fail for mixed states. This motivates the introduction of more sophisticated tools from functional analysis, specifically the nuclear norm and the concept of partial transpose. We present the Positive Partial Transpose (PPT) criterion as a powerful, computable test for entanglement. Furthermore, we explore its limitations by discussing PPT-entangled states, constructed using the notion of an Unextendible Product Basis (UPB), which necessitates even more general criteria for a complete characterization of entanglement.

Fundamental Concepts

We begin by recalling the basic definitions of states in finite-dimensional quantum mechanics.

Definition 1.1 (Quantum State) A quantum state $\rho$ is a positive semi-definite, self-adjoint operator on a complex Hilbert space $\mathcal{H}$ with unit trace. For these notes, we consider a finite $d$-dimensional space, $\mathcal{H} = \mathbb{C}^d$. A state must satisfy:

  1. $\rho = \rho^\dagger$ (self-adjoint)
  2. $\rho \ge 0$ (positive semi-definite)
  3. $\text{Tr}(\rho) = 1$ (unit trace)
The set of all such operators is denoted $\mathcal{M}_d(\mathbb{C})$.

Definition 1.2 (Pure State) A pure state is a state of rank one. It can be written as a projection operator $\rho = |\psi\rangle\langle\psi|$, where $|\psi\rangle \in \mathbb{C}^d$ is a unit vector, i.e., $\langle\psi|\psi\rangle = 1$.

Definition 1.3 (Mixed State) A mixed state is a state that is not pure (i.e., has rank greater than one). Any mixed state can be expressed as a convex combination of pure states: $$ \rho = \sum_i \lambda_i |\psi_i\rangle\langle\psi_i| \quad \text{with} \quad \lambda_i > 0, \sum_i \lambda_i = 1 $$

When dealing with a composite system, the total Hilbert space is a tensor product of the individual spaces, e.g., $\mathcal{H}_{AB} = \mathcal{H}_A \otimes \mathcal{H}_B = \mathbb{C}^{d_A} \otimes \mathbb{C}^{d_B}$. This structure gives rise to the phenomenon of entanglement.

Definition 1.4 (Separable State) A state $\rho$ on a composite Hilbert space $\mathcal{H}_A \otimes \mathcal{H}_B$ is separable if it can be written as a convex combination of product states: $$ \rho = \sum_i \lambda_i (\rho_i^A \otimes \rho_i^B) $$ where $\lambda_i > 0$, $\sum_i \lambda_i=1$, and each $\rho_i^A$ and $\rho_i^B$ are valid states on their respective subsystems. By Carathéodory's theorem, it is sufficient to consider the convex hull of pure product states, i.e., $\rho_i^A = |\psi_i^A\rangle\langle\psi_i^A|$ and $\rho_i^B = |\psi_i^B\rangle\langle\psi_i^B|$.

A state that is not separable is called entangled. Detecting entanglement is a central problem in quantum information theory.

The PPT Criterion for Entanglement

One of the most practical tools for detecting entanglement is the Positive Partial Transpose (PPT) criterion. It relies on applying the transpose operation to only one part of a composite system.

Definition 2.1 (Partial Transpose) The partial transpose of an operator $\rho$ on $\mathcal{H}_A \otimes \mathcal{H}_B$ with respect to subsystem $B$, denoted $\rho^{\Gamma_B}$, is defined by its action on product operators: $$ (A \otimes B)^{\Gamma_B} := A \otimes B^T $$ The definition is extended to all operators on $\mathcal{H}_A \otimes \mathcal{H}_B$ by linearity.

An important property is that the transpose of a density operator is also a density operator. If a state $\rho$ is separable, $\rho = \sum_i p_i \rho_i^A \otimes \rho_i^B$, then its partial transpose is $$ \rho^{\Gamma_B} = \sum_i p_i \rho_i^A \otimes (\rho_i^B)^T $$ Since each $(\rho_i^B)^T$ is a valid density operator, $\rho^{\Gamma_B}$ is a convex combination of density operators and is therefore itself a valid density operator. This means it must be positive semi-definite, $\rho^{\Gamma_B} \ge 0$. This leads to the famous Peres-Horodecki criterion.

Theorem 2.2 (Peres-Horodecki Criterion) If a state $\rho$ is separable, then its partial transpose $\rho^{\Gamma_B}$ is positive semi-definite. Consequently, if $\rho^{\Gamma_B}$ is not positive semi-definite (i.e., has at least one negative eigenvalue), then the state $\rho$ must be entangled.

This criterion can be reformulated using the nuclear norm, which provides a convenient numerical test.

Definition 2.3 (Nuclear Norm) The nuclear norm (or trace norm) of an operator $S$, denoted $\|S\|_*$, is the sum of its singular values $\sigma_i(S)$. The singular values are the square roots of the eigenvalues of $S^\dagger S$. $$ \|S\|_* = \text{Tr}\sqrt{S^\dagger S} = \sum_i \sigma_i(S) $$

For a positive semi-definite operator $A \ge 0$, its singular values are simply its eigenvalues, and the nuclear norm is equal to its trace: $\|A\|_* = \text{Tr}(A)$. If $\rho$ is a separable state, then $\rho^{\Gamma_B}$ is a valid density operator. This implies $\rho^{\Gamma_B} \ge 0$ and $\text{Tr}(\rho^{\Gamma_B}) = \text{Tr}(\rho) = 1$. Therefore, for any separable state, $\|\rho^{\Gamma_B}\|_* = 1$. The contrapositive gives us a powerful test.

Theorem 2.4 (Entanglement Test via Nuclear Norm) Let $\rho$ be a bipartite density operator. If the nuclear norm of its partial transpose is greater than one, $$ \|\rho^{\Gamma_B}\|_* > 1 $$ then the state $\rho$ is entangled.

Proof. Let $\rho$ be a separable state. Then it can be written as $\rho = \sum_i p_i \rho_i^A \otimes \rho_i^B$, where each $\rho_i^A, \rho_i^B$ is a density operator.
  1. The partial transpose is $\rho^{\Gamma_B} = \sum_i p_i \rho_i^A \otimes (\rho_i^B)^T$.
  2. Since the transpose of a density operator $(\rho_i^B)^T$ is also a density operator, $\rho^{\Gamma_B}$ is a convex combination of product density operators.
  3. This implies that $\rho^{\Gamma_B}$ is itself a valid density operator. It is therefore positive semi-definite ($\rho^{\Gamma_B} \ge 0$) and has trace 1.
  4. For any positive semi-definite operator $A$, its nuclear norm $\|A\|_*$ is equal to its trace $\text{Tr}(A)$.
  5. Therefore, $\|\rho^{\Gamma_B}\|_* = \text{Tr}(\rho^{\Gamma_B}) = 1$.
The contrapositive statement is that if $\|\rho^{\Gamma_B}\|_* > 1$, the state $\rho$ cannot be separable and must be entangled. □

Limits of the PPT Criterion: PPT-Entangled States

The PPT criterion is a powerful tool, but it is not a perfect one. It is a necessary condition for separability, but it is not sufficient in general. There exist entangled states that nonetheless have a positive partial transpose. These are known as PPT-entangled states or bound entangled states. A common way to construct such states is by using an Unextendible Product Basis (UPB).

Definition 3.1 (Unextendible Product Basis) An Unextendible Product Basis (UPB) is a set of orthonormal product vectors $\{|x_i\rangle = |a_i\rangle \otimes |b_i\rangle\}$ that span a proper subspace $\mathcal{E} \subset \mathcal{H}_A \otimes \mathcal{H}_B$, with the property that there is no other product vector $|\psi\rangle = |a\rangle \otimes |b\rangle$ that is orthogonal to every vector in the set. In other words, the orthogonal complement $\mathcal{E}^\perp$ contains no product vectors.

Using a UPB, we can construct a class of PPT-entangled states.

Proposition 3.2 (Werner-like states from UPBs) Let $\{|x_i\rangle\}_{i=1}^k$ be a UPB in a $d$-dimensional space $\mathcal{H}_A \otimes \mathcal{H}_B$. Let $P_{\mathcal{E}} = \sum_{i=1}^k |x_i\rangle\langle x_i|$ be the projector onto the subspace spanned by the UPB. The state defined by $$ \rho_{\mathcal{E}} = \frac{1}{d-k} (\text{Id} - P_{\mathcal{E}}) $$ is a PPT-entangled state (i.e., it is entangled but has a positive partial transpose).

Proof Sketch.
  • PPT Property: The partial transpose of $\rho_{\mathcal{E}}$ is $\rho_{\mathcal{E}}^{\Gamma_B} = \frac{1}{d-k}(\text{Id} - P_{\mathcal{E}}^{\Gamma_B})$. The partial transpose of the projector is $P_{\mathcal{E}}^{\Gamma_B} = \sum_i (|a_i\rangle\langle a_i| \otimes |b_i\rangle\langle b_i|)^{\Gamma_B} = \sum_i |a_i\rangle\langle a_i| \otimes |\bar{b_i}\rangle\langle\bar{b_i}|$. Since $\{|x_i\rangle\}$ are orthonormal, the set $\{|a_i\rangle \otimes |\bar{b_i}\rangle\}$ is also an orthonormal set of product vectors. Thus, $P_{\mathcal{E}}^{\Gamma_B}$ is also a projector. This means $\rho_{\mathcal{E}}^{\Gamma_B}$ is proportional to a projector $(\text{Id} - P_{\mathcal{E}}^{\Gamma_B})$ and is therefore positive semi-definite. So, the state is PPT.
  • Entanglement: A state is separable if and only if its support is spanned by product vectors. The support of $\rho_{\mathcal{E}}$ is the subspace $\mathcal{E}^\perp$. By the definition of a UPB, the subspace $\mathcal{E}^\perp$ contains no product vectors. Therefore, $\rho_{\mathcal{E}}$ cannot be separable and must be entangled.
□

Example: The "Tiles" UPB in $\mathbb{C}^3 \otimes \mathbb{C}^3$

A well-known example of a UPB is the "Tiles" basis, which consists of 5 vectors in a 9-dimensional space:

  • $|x_1\rangle = \frac{1}{\sqrt{2}}|0\rangle \otimes (|0\rangle - |1\rangle)$
  • $|x_2\rangle = \frac{1}{\sqrt{2}}|1\rangle \otimes (|1\rangle - |2\rangle)$
  • $|x_3\rangle = \frac{1}{\sqrt{2}}|2\rangle \otimes (|2\rangle - |0\rangle)$
  • $|x_4\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle) \otimes |2\rangle$
  • $|x_5\rangle = \frac{1}{\sqrt{3}}(|0\rangle+|1\rangle+|2\rangle) \otimes (|0\rangle+|1\rangle+|2\rangle)$

One can verify that these 5 vectors are orthonormal. The crucial part is to show they are unextendible. Suppose there exists a product vector $|\psi\rangle = |a\rangle \otimes |b\rangle$ such that $\langle x_i | \psi \rangle = 0$ for all $i=1, \dots, 5$. This leads to a set of linear equations on the components of $|a\rangle$ and $|b\rangle$ that has no non-trivial solution, proving unextendibility. The corresponding state $\rho = \frac{1}{4}(\text{Id}_9 - \sum_{i=1}^5|x_i\rangle\langle x_i|)$ is therefore a PPT-entangled state.

Toward a General Criterion for Entanglement

The existence of PPT-entangled states shows that the PPT criterion is not a complete solution to the entanglement detection problem. The ultimate goal is to find a condition that is both necessary and sufficient for separability. This leads us to a more geometric and functional-analytic perspective.

The Geometry of Separable States

The set of all quantum states $\mathcal{D}$ is a convex set. The subset of separable states, which we denote by $S$, is also a convex set. Specifically, $S$ is the convex hull of all pure product states. $$ S = \text{conv} \{ |\psi\rangle\langle\psi| : |\psi\rangle = |\psi_A\rangle \otimes |\psi_B\rangle \otimes \dots \}$$ A state $\rho$ is separable if and only if $\rho \in S$. All other states in $\mathcal{D} \setminus S$ are entangled. This geometric picture suggests that we can test for entanglement by determining if a state lies inside or outside the convex set $S$.

Functional analysis provides a tool for this: the Minkowski functional (or gauge function) of a convex set.

Definition 4.1 (Minkowski Functional) For a convex set $K$ containing the origin, its Minkowski functional $p_K(x)$ is defined as: $$ p_K(x) = \inf \{ \lambda > 0 : x \in \lambda K \} $$ Under suitable conditions (e.g., $K$ is closed and bounded), the set $K$ can be recovered from its functional as $K = \{ x : p_K(x) \le 1 \}$.

Applying this to our problem, if we can define a functional $p_S(\rho)$ for the set of separable states $S$, then a state $\rho$ would be separable if and only if $p_S(\rho) \le 1$. This would provide the desired "if and only if" criterion. This leads to the idea of tensor norms. A suitable norm $\|\cdot\|$ would have the property that $\|\rho\| \le 1$ for all separable states $\rho$. Then, if we find a state with $\|\rho\| > 1$, it must be entangled.

The appropriate norm for this task is the projective tensor norm.

Definition 4.2 (Projective Cross Norm) The projective norm of an operator $\rho$ on a multipartite space is defined via its decompositions into a sum of product operators: $$ \|\rho\|_{\pi} = \inf \left\{ \sum_i |c_i| : \rho = \sum_i c_i (\rho_i^{(1)} \otimes \dots \otimes \rho_i^{(n)}) \right\} $$ where the infimum is taken over all possible decompositions of $\rho$ into pure product states $\rho_i^{(j)}$.

By its very definition, if a state $\rho$ is separable, it is a convex combination of product states (i.e., $c_i=p_i > 0$ with $\sum p_i = 1$), so its projective norm is $\|\rho\|_\pi \le \sum_i p_i = 1$. This provides the complete theoretical solution.

Theorem 4.3 (Separability via Projective Norm) A multipartite state $\rho$ is separable if and only if its projective norm is less than or equal to one. $$ \rho \text{ is separable } \iff \|\rho\|_{\pi} \le 1 $$ Equivalently, $\rho$ is entangled if and only if $\|\rho\|_{\pi} > 1$.

While this provides a complete theoretical answer, computing the projective norm is an NP-hard problem in general, making it impractical for direct application. The search for computable and powerful entanglement criteria remains an active area of research.
